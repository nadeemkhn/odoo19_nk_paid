Leopards Courier Integration

Odoo 16 Shipping Automation

Integrate Odoo Delivery with Leopards Courier API for live rates, shipment booking, labels, tracking updates, and cancellations from within Odoo.

Why This Module

  • Direct Leopards API integration for operational speed.
  • No manual courier portal entry for each shipment.
  • Clear workflow: validate transfer, then book shipment.
  • Built for Odoo 16 delivery and website sales flow.

Key Features

  • Delivery Method Type: Adds leopards carrier type.
  • Live Rates: Uses getTariffDetails for shipping cost.
  • Booking: Uses bookPacket from delivery orders.
  • Tracking: Uses trackBookedPacket with status sync.
  • Cancellation: Uses cancelBookedPackets.
  • Shipper Management: Dedicated shipper records and per-picking override.
  • Automation: Cron jobs for tracking refresh and queued cancellations.

Configuration

  1. Go to Inventory > Configuration > Delivery Methods.
  2. Create/edit method and set Delivery Type = Leopards Courier.
  3. Set API Key, API Secret, API URL, and optional Account ID/Shipper.
  4. Click Test Connection and save.

Operational Flow

  1. Create quotation/sales order and add Leopards shipping method.
  2. Get shipping rate and confirm the order.
  3. Validate the delivery transfer.
  4. Open delivery and click Book a Shipment.
  5. Track CN status and cancel from Odoo if required.
