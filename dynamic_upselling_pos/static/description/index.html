Dynamic Upselling POS

Add smart upselling suggestions in Odoo Point of Sale. Configure products per POS shop, and the popup appears when clicking Payment to quickly add extra items.

How To Use

  1. Go to Point of Sale - Configuration - Dynamic adding Products.
  2. Create a rule and select the Shop(s) where upselling should apply.
  3. Select the Products you want to suggest in POS.
  4. Open a POS session and add items to the order.
  5. When you click Payment, the upselling popup appears. Click Add, then Okay or Skip.
