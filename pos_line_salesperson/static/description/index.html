POS Line Salesperson

Assign salesperson per POS order line, restrict selectable salespersons by shop, and analyze salesperson performance directly from POS reports.

Features

  • Salesperson field on each POS order line.
  • Salesperson shown in POS line UI and backend order lines.
  • Shop-level employee selection for salesperson button.
  • Employee avatar in salesperson selection popup.
  • Salesperson support in POS Orders Analysis search/grouping.

How It Works

  • Configure allowed employees in Point of Sale Configuration.
  • Open POS session and select any order line.
  • Click Salesperson button and choose employee.
  • Salesperson is saved per line and available in reporting.
